Now your Firefox browser has the function to inject furigana, the small hiragana letters above or beside difficult kanji words. Try it - click the icon in the statusbar.

What should I do first?

Install the HTML Ruby extension

(A link to mozilla.org's page for the add-on developed by kailapis. Please note there is no problem if you use Shimoda Hiroshi's XHTML Ruby Support add-on instead.)

With ruby support:
とうきょうおおさか
東京から大阪までの...
Without ruby support: 東京（とうきょう）から大阪（おおさか）までの...

Select your kanji-reading level

Choose the kanji you do and don't want furigana for.

Set your kanji-reading level

Use the "Options..." (see below) to set the list of the kanji you don't need furigana for. Note: if you have already installed the HTML Ruby addon there is a quick and easy Kanji-selecting page you can use instead.


Then use this tool in any web page that has japanese text. www.mozilla.gr.jp is a good example that shows:

  • Words composed of simple kanji won't be processed at all.
  • Words in links wont be processed, unless you change the preference for that.
  • Images are ignored so text in graphics won't be changed, sorry.

Privacy notice

Please note that from version 2.0 the text is processed on a remote server. Simply do not use Furigana Injector on pages or paragraphs or sentences that have personal information.


Other options ...

  • You can choose to add furigana to text in hyperlinks as well as normal text.
  • Pop-up translations from WWWJDIC can be shown when mousing-over furigana.

These options can be accessed directly by right-clicking on the icon.
